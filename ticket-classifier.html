IT Support Ticket Classifier

A DistilBERT-powered NLP model that classifies IT help-desk tickets into predefined issue categories.

Overview

This project builds a machine learning system that automatically classifies IT support tickets. Using a fine-tuned DistilBERT model from HuggingFace Transformers, it turns unstructured help-desk text into meaningful categories such as: Email Issues, Networking, Hardware, Software Access, and more.

How It Works

  • Loads raw help-desk text from CSV
  • Uses DistilBertTokenizerFast to convert text into token IDs
  • Fine-tunes DistilBERT for 3 epochs using a classification head
  • Evaluates using accuracy + cross-entropy loss
  • Exports final model, tokenizer, and label encoder

Training Details

  • Model: distilbert-base-uncased
  • Batch Size: 8
  • Learning Rate: 5e-5
  • Epochs: 3
  • Framework: PyTorch + HuggingFace Transformers

Example Prediction

Input:
"My Outlook keeps crashing whenever I open an email."

Prediction:
"Email Issue"

Next Steps

I plan to deploy this model using FastAPI and build a simple web UI where users can paste ticket text and instantly view predictions. This will mimic real enterprise IT ticket systems.
