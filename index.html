Featured Project: AI Incident-Triage Assistant (‚ÄúJARVIS‚Äù)

A custom-built AI IT helpdesk assistant that classifies issues, runs troubleshooting flows, and generates escalation-ready tickets.

I engineered an AI-powered IT support assistant that behaves like a real service desk technician. It uses a fine-tuned DistilBERT classifier I trained on IT ticket data to predict issue categories, then routes users into 20+ modular troubleshooting flows (VPN, Wi-Fi, Outlook, printers, onboarding, hardware, etc.).

The system maintains conversation state (device, OS, location, symptoms, last flow), walks users through multi-step diagnostics with a JARVIS-style persona, and exports complete incident tickets in JSON, text, or Markdown.

Architecture & Tech Stack
  • Model: Custom fine-tuned DistilBERT (PyTorch/Transformers)
  • Backend: Python, Gradio, HuggingFace Spaces
  • Logic: State machine for 20+ troubleshooting flows
  • Output: JSON/Markdown ticketing with auto-escalation levels
